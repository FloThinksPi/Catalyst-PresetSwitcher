Catalyst Preset Switcher

Switch your CatalystControlCenter Presets remotely from your Android Phone

Welcome to Catalyst Preset Switcher

The App needs the Server Programm running on the target Windows Computer, additionally both devices(PC and Phone) need to be in the same Network. Port Forwarding is not recomendet because this software does not offer any security features.

Requirements:
-Windows Machine
-AMD Graphics Card with Catalyst Controll Center Installed
-Android Phone(JellyBean or newer)
-!WORKING PRESETS!(Catalyst Presets often do not work for some Reason , if they dont work on  your pc manually , they wont also work remotely with your phone so test them)

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
